# Javascript_Basic

A small learning repository for JavaScript fundamentals (and related concepts), organized by chapter.

## Contents

- `Chapter01_Basics.js`
- `Chapter02_Java_Concepts/`

## How to run

This repo currently contains plain JavaScript files.

1. Install Node.js (LTS recommended)
2. Run a chapter file:

```bash
node Chapter01_Basics.js
```

## Notes

- As new chapters are added, they will follow the same naming pattern.
